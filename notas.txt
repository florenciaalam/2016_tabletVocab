EDICION DE IMAGENES, ULTIMAS VERSIONES
    Removing background - with GIMP : Layer menu --> transparency --> add an alpha channel --> shift + O + clic on the background --> erase or suppr --> save --> need to convert it in png (for example http://image.online-convert.com/fr/convertir-en-png)
    - with PowerPoint (when it is complicated with GIMP) : double clic on the picture --> remove the background (top left) --> mark areas to keep and delete (the pink areas will be deleted by clicking next to the image) --> after that, right clic on the picture --> save as picture (.png)

    Add a background with PowerPoint : first remove the background --> add a desired color rectangle and put the picture on it --> screen print (on the keyboard) --> right clic --> paste --> doucle clic on it --> crop the image --> right clic on it --> save as picture (.png)

No need to change the size of the images before add them on iDevXXI. However, if the picture is too big, you can add some blank around it with Paint.
